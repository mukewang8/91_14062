<h3>课程介绍：</h3>
<p>百战商城项目是一个千万级综合性的B2C电子商务平台模块。用户可以在此系统中实现搜索商品、查看商品详情、将商品加入购物车、购买商品、生成订单、完成购物等一系列操作。开发此项目，我们采用前后端分离架构的方式，前端使用Vue.js开发，后端使用微服务架构方式开发，同时运用目前非常火爆的SpringCloud作为服务治理平台</p>
